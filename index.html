📖 How to Use This Dashboard — Complete Guide

📂 Updating Data via CSV

  • Export your KoboToolbox form as CSV
    (Data → Downloads → CSV)
  • Click "CSV Update" in the header
  • Paste the CSV text or upload the file
  • Click "Load CSV" — dashboard refreshes instantly
  • Required columns: district, block, partner
  • All other columns are auto-detected

🔗 Live Auto-Refresh (KoboToolbox API)

  • Click "Live API" in the header
  • Enter your API Token: KoboToolbox → Account → Security
  • Enter your Asset UID: from your form's URL
  • Set auto-refresh interval (e.g. every 5 minutes)
  • Click "Connect" — dashboard auto-updates after every new submission
  • ⚠ Requires the file to be hosted on a web server (not opened locally) due to browser CORS rules

⚙ Adding / Removing Indicators

  • Click "Indicators" in the header
  • Check or uncheck any KPI card or chart
  • Changes apply immediately — no reload needed
  • To add a new custom indicator, use "Add Custom KPI" and enter the CSV column name
  • Your indicator settings are remembered during the session

🔍 Filtering Data

  • Use the District / Partner / Block dropdowns to filter all charts simultaneously
  • Select "All" to reset any filter
  • Filters work on both CSV and live API data

📋 CSV Column Name Guide

  • district — District name
  • block — Block name
  • partner — Partner organisation
  • cm_prevented — Child marriages prevented
  • vuln_m / vuln_f — Vulnerable boys/girls
  • oos_girls / oos_boys — Out-of-school
  • pcwpc — yes/no — PCWPC formed
  • ch1098 / pol112 / wh181 — Helplines painted (yes/no)

💡 Tips

  • The dashboard works fully offline — no internet needed for CSV mode
  • Click any bar in a chart to see a tooltip with exact values
  • The built-in data (17 rows) is always available as a fallback
  • You can share this HTML file with colleagues — it contains everything
  • For questions, contact your data team or check KoboToolbox documentation
